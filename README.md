# EMU Swap - Flow Network
## Getting Started
### You can test using emulator

First, run the development server:

```bash
npm run dev
# or
yarn dev
```
Next, run flow emulator and dev-wallet:

```bash
cd contracts
flow emulator
flow dev-wallet
```
Then, run setup.sh file to initialize

```bash
./bash/setup.sh
```
Server is starting on http://localhost:3000
